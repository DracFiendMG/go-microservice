A simple microservice using Go Lang where you can add a user and view the list of users

Microservice Structure:

```
go-microservice/
├── main.go
├── router/
│   └── router.go
├── handler/
│   └── user_handler.go
├── model/
│   └── user.go
├── config/
│   └── config.go
└── database/
    └── database.go
```
